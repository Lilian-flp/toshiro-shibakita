<h1> Docker </h1>
<p>Docker é uma plataforma de software que permite criar, testar e implementar aplicativos de forma rápida e fácil, usando "contêineres". Imagine contêineres como caixas que contêm tudo o que um aplicativo precisa para ser executado: código, bibliotecas, ferramentas e configurações.</p>
